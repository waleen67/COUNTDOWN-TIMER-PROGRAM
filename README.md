# ⏳ Countdown Timer in Python

This is a simple countdown timer written in Python. The program takes input time in seconds and counts down to zero, displaying the remaining time in HH:MM:SS format.

## 🚀 Features
- Accepts time input in seconds
- Displays countdown in hours, minutes, and seconds
- Updates every second
- Prints "TIME'S UP!" when finished

## 🛠️ Requirements
- Python 3.x
